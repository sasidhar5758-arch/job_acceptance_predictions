# Job Acceptance Prediction System

## Project Overview

The Job Acceptance Prediction System is a Machine Learning project designed to predict whether a candidate will accept or reject a job offer based on academic performance, skills, work experience, interview scores, and company-related factors.

This project helps recruitment teams reduce offer dropouts, improve hiring efficiency, and make data-driven recruitment decisions.

---

## Problem Statement

Recruitment teams often face challenges when candidates reject job offers after receiving them. This project analyzes candidate data and builds a predictive model to estimate the probability of job acceptance.

---

## Objectives

* Predict whether a candidate will accept a job offer.
* Identify factors influencing job acceptance.
* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA).
* Build and evaluate Machine Learning models.
* Create an interactive Streamlit dashboard.
